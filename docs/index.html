    JSMailSender Reference

    Intro

    JSMailSender is a simple SMTLP client for sending an email from an iOS or Mac OS X application. It is designed for sending an occasional email to a single recipient. This works well for error reporting, feedback screens, and other low-volume use.

    Usage

    The only object you need to instantiate is JSSMTPConnection. The following properties must be set for that class to work:

    • relayHost
    • port
    • senderEmail
    • recipientEmail
    • message

    If you expect your server to require authentication (which is almost always), you also need to set

    • username
    • password

    Once you have set those properties, send a begin message and it will open a connection, send the message, and close the connection. Once the connection is closed, you can’t open it again and you must create another JSSMTPConnection to send another mail. The other public methods are generally for use by the finite state machine that drives the connection, though you could use them if you have some specific need.
